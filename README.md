## Tech Stack

- ⚛️ React 19 + TypeScript
- ⚡ Vite
- 🎨 Tailwind CSS
- 🎞️ Framer Motion

## Features

- 🖥️ Interactive CLI emulator
- 📊 Algorithm visualizer playground
- 🗂️ Project showcase with filtering
- 🪪 Developer identity card
- 📅 Professional journey timeline
- 📬 Contact form

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
npm run deploy
```
